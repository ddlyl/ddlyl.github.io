什么是数据挖掘

Data Mining is the process of automatically extracting interesting and useful hidden patterns from usually massive, incomplete and noisy data.

过程

  • 输入数据
  • 预处理
    • 特征选择
    • 维归约
    • 规范化
    • 选择数据子集
  • 数据挖掘
  • 后处理
    • 模式过滤
    • 可视化
    • 模式表示
  • 信息

要解决的问题

  • 可缩性(scalable) ：处理容量很大的数据
  • 高维性：维度(特征数)的增长正比于测量次数
  • 异种数据和复杂数据
  • 数据的所有权与分布
  • 非传统的分析

数据挖掘的任务

  • 预测任务
  • 描述任务：找到潜在联系模式
  • 预测建模
    • 分类
    • 回归
  • 关联分析：尿布->牛奶
  • 聚类分析：发现紧密相关的观测值组群
  • 异常检测

数据

定量或者定性的属性值

数据经过处理成为信息

大数据

  • high-volume
  • high-velocity：数据源源不断来
  • high-variety：种类多，非结构化
  • beyond the ability of typical database software tools to capture, store, manage, and analyze

分类

通过过去的学习来预测物体的属性

数据分成训练集和测试集，用训练集生成模型，用测试机验证模型

Confusion Matrix

$$TPR = \frac{TP}{TP+FN}$$
$$TNR = \frac{TN}{TN+FP}$$
$$Accuracy = \frac{TP+TN}{P+N}$$

AUC曲线


横轴为FP值，纵轴为TP值

假设我们通过身高区别男人：

  1. 身高高于1m的都是男人：则无论男人女人都被判定为男人，此时TP = 1，FP= 1，在图像最右上
  2. 身高高于2.5m的都是男人：则无论男人女人都被判定为女人，此时TP= 0，FP= 0，在图像最左上。
  3. 因此当我们选择的判定身高介于两者之间，会出现在图中的某个位置。$y = x$的直线表示机器随机猜得到的结果，我们希望我们的判定曲线可以尽量的往上走，因此希望他的面积越大越好，定义为AUC（Area Under Curve）。值越接近于1越好

注：错误的代价成本是不一样的，如误会好人和放过坏人
