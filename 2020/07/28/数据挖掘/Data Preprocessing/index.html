    熵

$$H(X)=-\sum_{i=1}^{n}p\left(x_{i}\right)\log {b}p\left(x{i}\right)$$

$H(S) = -0.5 \cdot \log _{2} 0.5-0.5 \cdot \log _{2} 0.5 = 1.0$
$X:{a = “ \text { Non }-\text { Smoker”; } b = “ \text { Smoker” }}$
$H(S \mid X = a) = -0.8 \cdot \log _{2} 0.8-0.2 \cdot \log _{2} 0.2 = 0.7219$
$H(S \mid X = b) = -0.05 \cdot \log _{2} 0.05-0.95 \cdot \log _{2} 0.95 = 0.2864$
$H(S \mid X) = 0.6 \cdot H(S \mid X = a)+0.4 \cdot H(S \mid X = b) = 0.5477$
$\operatorname{Gain}(S, X) = H(S)-H(S \mid X) = 0.4523 \ 信息增益,越大越好$
