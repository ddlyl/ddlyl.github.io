会计核算基础知识

会计要素与会计等式

会计要素

会计要素是会计对象的具体化，是对会计核算和监督内容的基本分类

  • 资产
  • 负债
  • 所有者权益（净资产）

反应一定日期财务状况

  • 收入
  • 费用
  • 利润

反应一定时期经营成果要素

会计等式

资产 = 权益

资产 = 负债 + 所有者权益

收入 - 费用 = 利润

资产 = 负债 + 所有者权益 + （收入 - 费用）

会计科目和账户

会计科目

会计科目对会计要素的具体内容进一步分类的项目名称

  1. 资产类
  2. 负债类
  3. 所有者权益类
  4. 费用类
  5. 损益类

按照核算的详略程度不同，分为总分类科目和明细科目

账户

账户是根据会计科目开设的，具有一定的格式和结构，用于分类、系统、连续的纪录经济业务的一种工具

账户应该包括的基本内容：

  1. 账户名称，即会计科目
  2. 经济业务发生的日期
  3. 据以等级的会计凭证号数
  4. 经济业务的内容摘要
  5. 增加或减少的金额

复式记账法

预收收入不是收入是负债

预付费用不是费用是资产

理论依据：会计等式

能够全面、清晰的反应经济业务的全貌

借方 贷方
资产类账户 + -
负债及所有者权益 - +
费用类账户 + -
收入类账户 - +
分红 - +

​ 所有者权益

= 实收资本 + 留存收益

= 实收资本 + 期初留存收益 + 当期净收益 - 分红

= 实收资本 + 期初留存收益 + 收入 - 费用 - 分红

有借必有贷，借贷必相等

  • 借贷记账法：
  1. 对每一笔经济业务都必须同时计入两个或两个以上相互联系的账户，并且以借贷相反的方向登记入账，使得经济业务之间形成应借，应贷的相互依存关系
  2. 每一项经济业务计入借方的金额同计入贷方的金额必须相等
  • 试算平衡

  1. 每笔会计分录的试算平衡

    借方科目金额 = 贷方科目金额

  2. 全部账户本期发生频额试算平衡

    全部账户借方发生额合计 = 全部账户贷方发生额合计

  3. 全部账户期末余额试算平衡

    全部账户借方余额合计 = 全部账户贷方余额合计

举例

  1. 借：银行存款 5000

    贷：现金 5000

  2. 借：材料 20000元

    贷：应付账款 20000元

  3. 借：贷款50万

    贷：银行存款50万

  4. 借：短期贷款20万

    贷：长期贷款20万

  5. 借：应付账款8000

    贷：银行存款8000

  6. 借：固定资产25万

    贷：银行存款25万

会计循环

基本程序

  • 期末账项调整

    1. 收入项目调整：登记应收收入，调整预收收入
    2. 费用项目调整：登记应计费用，调整预付费用

  • 对账与结账

    • 对账: 在过账之后，保证账簿纪录的正确性。账证核对，账账核对，账实核对
    • 结账：结转本期的成本与费用，结转本期实现的各项收入和利得，结转其他账户

  • 试算平衡
