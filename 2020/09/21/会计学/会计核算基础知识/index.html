会计核算基础知识

会计要素与会计等式

会计要素

会计要素是会计对象的具体化，是对会计核算和监督内容的基本分类

  • 资产
  • 负债
  • 所有者权益（净资产）

反应一定日期财务状况

  • 收入：企业在日常活动中形成的，会导致所有者权益增加的、与所有者投入资本无关的经济利益的总流入（非日常活动——利得，如出售固定资产）
  • 费用：企业在日常活动中形成的，会导致所有者权益减少的、与所有者分配利润无关的经济利益的总流出（非日常活动——损失，如捐赠）

反应一定时期经营成果要素

会计等式

资产 = 权益

资产 = 负债 + 所有者权益

收入 - 费用 = 利润

资产 = 负债 + 所有者权益 +利润

资产 + 费用 = 负债 +所有者权益 + 收入

常见情形

  • 收回贷款： 资产内部增减，总额不变
  • 偿还贷款： 资产总额和负债总额同减
  • 预收货款： 资产总额和负债总额同增
  • 预付货款： 资产内部增减，总额不变
  • 开出银行汇票未交付：资产内部增减，总额不变
  • 收到银行汇票：资产总额增加

会计科目和账户

会计科目

会计科目对会计要素的具体内容进一步分类的项目名称

  1. 资产类
  2. 负债类
  3. 所有者权益类
  4. 费用类
  5. 损益类

按照核算的详略程度不同，分为总分类科目（应交税费）和明细科目（应交所得税）

账户

账户是根据会计科目开设的，具有一定的格式和结构，用于分类、系统、连续的纪录经济业务的一种工具

账户应该包括的基本内容：

  1. 账户名称，即会计科目
  2. 经济业务发生的日期
  3. 据以等级的会计凭证号数
  4. 经济业务的内容摘要
  5. 增加或减少的金额

复式记账法

预收收入不是收入是负债

预付费用不是费用是资产

理论依据：会计等式

能够全面、清晰的反应经济业务的全貌

费用 + 资产（成本） = 负债 + 所有者权益 + 收入（大拇指法）

特殊资产账户（累计、准备）：记账方向与资产账户相反

收入、费用类账户期末余额

借方 贷方
资产类账户 + -
负债及所有者权益 - +
费用类账户 + -
收入类账户 - +
分红 - +

​ 所有者权益

= 实收资本 + 留存收益

= 实收资本 + 期初留存收益 + 当期净收益 - 分红

= 实收资本 + 期初留存收益 + 收入 - 费用 - 分红

有借必有贷，借贷必相等

  • 借贷记账法：
  1. 对每一笔经济业务都必须同时计入两个或两个以上相互联系的账户，并且以借贷相反的方向登记入账，使得经济业务之间形成应借，应贷的相互依存关系
  2. 每一项经济业务计入借方的金额同计入贷方的金额必须相等
  • 试算平衡

  1. 每笔会计分录的试算平衡

    借方科目金额 = 贷方科目金额

  2. 全部账户本期发生频额试算平衡

    全部账户借方发生额合计 = 全部账户贷方发生额合计

  3. 全部账户期末余额试算平衡

    全部账户借方余额合计 = 全部账户贷方余额合计

举例

  1. 借：银行存款 5000

    贷：现金 5000

  2. 借：材料 20000元

    贷：应付账款 20000元

  3. 借：贷款50万

    贷：银行存款50万

  4. 借：短期贷款20万

    贷：长期贷款20万

  5. 借：应付账款8000

    贷：银行存款8000

  6. 借：固定资产25万

    贷：银行存款25万

会计循环

基本程序

期末账项调整

  1. 收入项目调整：登记应收收入，调整预收收入
  2. 费用项目调整：登记应计费用，调整预付费用

对账与结账

  • 对账: 在过账之后，保证账簿纪录的正确性。账证核对，账账核对，账实核对
  • 结账：结转本期的成本与费用，结转本期实现的各项收入和利得，结转其他账户

试算平衡

  1. 发生额试算平衡：全部借方 = 全部贷方，不看具体科目

    依据：有借必有贷，借贷必相等

  2. 余额试算平衡：全部账户借方期末（初）月合计与全部账户贷方期末（初）月合计保持平衡

    依据： 财务状况等式

    资产 = 负债 + 所有者权益

    借余 = 贷余 + 贷余

  • 不影响借贷双方平衡关系的错误：
    1. 重记经济业务（会计科目x）
    2. 漏记
    3. 颠倒方向
    4. 记错账户
    5. 双方金额多记或者少记
    6. 偶然多记和少记并相互抵消

会计凭证、会计账簿

会计凭证

分类

按照填制的程序和用途不同，分为原始凭证（业务发生时的证明，原始依据）和记账凭证（会计人员根据审核无误的原始凭证，确定会计分录后所填制的会计凭证，事登记会计账簿的直接依据）

累计凭证-限额表单

原始凭证基本内容

名称、日期、单位名称姓名、接受贷证单位名称、经济业务内容（交易或事项内容）、数量、单价、金额、经办人员签名或盖章

  • 不属于原始凭证基本内容的：借贷方向，会计科目，记账符号，附原始凭证的张数

原始凭证填制要求

  • 不得涂改、刮擦、挖涂

  • 原始凭证有金额错误，应由出具单位重开，不得在原始凭证上更正

  • 原始凭证有其他错误，应由出具单位重开或更正，更正处应该加盖出具单位印章

记账凭证

收款凭证—— 纪录现金和银行存款 收款 业务的记账凭证

付款凭证—— 纪录现金和银行存款 付款 业务的记账凭证

转账凭证——纪录不涉及现金和银行村矿业务的记账凭证

对于涉及“库存现金”和“银行存款”之间的经济业务一般只编制付款凭证，不编制收款凭证

出纳员办理完收款业务后，应在原始凭证上加盖“收讫”

记账凭证的填制要求

会计账簿

分类

  1. 按用途

    序时账簿（逐日逐笔），分类账簿（总分类和明细分类账簿），备查账簿

  2. 按格式

    三栏式：借贷余额，日记账，总账以及资本、债权、债务明细账

    多栏账：收入、成本、费用、利润分配明细账（日记账有时也可以）

    数量金额帐：原材料、库存商品明细账

  3. 按外形

    订本式：不易丢，但是不能准确为账户预留账页；总分类账，库存现金日记账，银行存款日记账

    活页式：方便加减，便与分工记账；各种明细分类账

    卡片式：固定资产核算

  1. 总分类账和明细分类账的平行登记

    平行登记，是指对所发生的每项经济业务都要以会计凭证为依据，一方面计入总分类账户，另一方面计入所属的明细分类账户的方法（依据相同）

    • 方向相同
    • 期间一致（具体时间可以有先有后，但要在一个会计期间内完成）
    • 金额相等

对账与结账

  1. 对账

    对账是对账簿记录所进行的核对，就是核对账目

    一般分为账证核对、账账核对和账实核对

    对账工作一般在记账之后结账之前，即在月末进行

  1. 结账
    • 结清各种损益类账户，并据以计算确定本期利润
    • 结出各资产、负债和所有者权益的本期发生额合计和期末余额

错账更正

  1. 划线

    在结账前发现账簿记录有文字或数字错误，而记账凭证没有错误，采用划线更正法

  2. 红字更正法

    1. 应借应贷会计科目有错误
    • 先用红字填写一张与原记账凭证完全相同的记账凭证，并在摘要栏注明“冲销X月X日X号凭证错账”，并据以红字登记入账，以示注销原记账凭证
    • 其次用蓝字在编写一张正确的记账凭证，并据以用蓝字登记入账。（并在在摇篮注明“补记X月X日X号凭证”）
    1. 应借、应贷会计科目无误，所记金额大于应记金额
    • 按多记用红字编制一张
    1. 应借、应贷会计科目无误，所记金额小于应记金额
    • 按少记用蓝字编制一张

账务处理程序

  1. 记账凭证账务处理程序

    特点：直接根据记账凭证逐笔登记总分类账

    优点：简单明了，可以较详细的反应经济业务的发生情况

    缺点：登记总分类账的工作量较大

    规模小的企业，经济业务较小的企业

  2. 汇总记账凭证账务处理程序

  3. 科目汇总表账务处理程序

科目汇总表只反应各个会计科目借方本期和贷方本期发生额，不反应各个会计科目的对应关系

  • 主要区别：登记总分类账的依据和方法不同
